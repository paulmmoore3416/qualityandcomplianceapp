## üìÑ Templates

The `/templates` folder includes ready-to-use quality system templates:

### Available Templates

- **Risk Assessment Template** (ISO 14971) - Excel format
- **CAPA Form** (ISO 13485) - Word format
- **NCR Template** - Word format
- **Change Control Form** (21 CFR Part 11) - Word format
- **Audit Checklist** (ISO 13485) - Excel format
- **Supplier Audit Checklist** - Excel format
- **Training Record Template** - Excel format
- **Document Control Log** - Excel format
- **Design History File (DHF) Template** - Word format
- **Device Master Record (DMR) Template** - Word format

### Using Templates

1. Templates are provided as starting points
2. Customize for your organization's specific processes
3. Validate templates per your QMS procedures
4. Maintain version control per document control procedures
